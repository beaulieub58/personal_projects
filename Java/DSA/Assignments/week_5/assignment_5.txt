(7.2) What is the running time insertion sort if all elements are equal?
    
     In the insertion sort algorithm provided in the text, if all elements of the array are equal,
     the inner for loop will not initiate and fail. This leads us to our best case
     in time complexity for insertion sort, which is O(N).

     The inner loop condition will fail if the array is presorted due to the second condition in
     in the inner for loop (denoted by ** // **): 
     for( j = p; j > left && **tmp.compareTo( a[ j - 1 ] ) < 0**; j-- )
